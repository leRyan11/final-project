Implementation Plan

Feature 1: Snake Movement
Trigger: Player presses arrow keys  
The snake moves in the chosen direction each frame.

Feature 2: Collect Clipper Cards
Trigger: Snake reaches card  
The snake grows and score increases.

Feature 3: Maps
Trigger: Game start  
Loads a themed BART station map.

Feature 4: Collision Detection
Trigger: Snake hits wall or itself  
Game ends.

 Feature 5: Save Score
Trigger: Game ends  
Saves high score to file.
